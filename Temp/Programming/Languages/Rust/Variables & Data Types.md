> [!tip] 
> - All variables are immutable by default.
> - Variable type can be specified or inferred.

### Numbers

```rust
let x: i32 = -48;
let y: u8 = 10;
let z: f16 = 1000.001;
```

`i32` = 32-bit signed integer
`u` -> unsigned integer
`f16` -> float

### Bool

```rust
let peace: bool = true;
```

### String

```rust
let greeting: String = String::from("Hello World");
```
